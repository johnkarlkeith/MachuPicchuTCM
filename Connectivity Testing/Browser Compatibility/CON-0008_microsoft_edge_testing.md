## **CON-0008:** Microsoft Edge testing  

> **Summary:** Verify the app is working on Microsoft Edge browser.  <br>

**Preconditions:**   

 - Latest version of Microsoft edge is installed.
 - WebApp URL is accessible.

Scenario 1 

 | \# | Step | Expected Behavior | 
 |----|------|-------------------| 
 |  1 | Launch Microsoft edge and open the WebApp URL.  | WebApp loads without errors   | 
 |  2 | Log in and navigate to key features.            | Navigation, UI, and interactivity work smoothly.   | 
 |  3 | Perform data actions (add/edit/delete).         | Forms and buttons respond and save data successfully.   |
 |  4 | Check visual elements.                          | Fonts, icons, and layouts are properly aligned.   |    

**Post-conditions:**  

 - App is fully functional on Microsoft edge with correct display and performance.
